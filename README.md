# 🚍 Bus Ticket Booking System

**Client-Server Bus Ticket Booking Application** using JavaFX and TCP Sockets.

## Team Members
- Abdulrahman Keshk
- Abdulrahman Essam

## Features
- User Registration & Login
- Browse Available Trips
- Real-time Seat Booking
- View My Bookings
- Profile Management
- Multithreaded Server

## Technologies
- Java + JavaFX (GUI)
- TCP Socket Programming
- Multithreading (ExecutorService)
- OOP Principles
- Custom Exception Handling

## How to Run
1. Run the **Server** first (`network.Server`)
2. Run the **Client** (`client.MainApp`)
